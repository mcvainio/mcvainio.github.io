Web GIS Examples
UNDER CONSTRUCTION
The goal of this collection is to provide a set of instructive examples which introduce a variety of open source plug-ins. Some of these plug-ins are designed specifically for web mapping, while others are more general. The source code for each page contains detailed comments.
Hosted at GitHub. Main page css taken from Stemkoski's Three JS example page (thank you!!). Last updated: 27 Nov 2022.

Hello World

A heavily commented but basic scene. Illustrates the setup of a scene, camera, renderer.


Leaflet - Query Third-Party WMS

Uses a click listener on a Leaflet map to grab a user-selected co-ordinate. Then, fires a single query at a geoserver using that co-ord. The result (the depth property of the WMS) is displayed in an inline window on the map.


Info Box

The template code with minimal jQuery added to create a button that displays an information dialog box.


Materials - Solid

Illustrates the difference between the Basic, Lambert, and Phong materials.


ThreeJS - Plain Plane

Shows how to generate a wireframe mesh of a plane. Our subsequent examples will use this plane as a base, modifying it to model real-world terrain.


Texture - Repeat

Repeating (tiling) a texture on a surface.


Texture - Text3D

Applying a texture to 3D Text. (Requires repeating texture on surface.)


Translucence

Illustrates using basic and phong-shaded translucent materials, making image textures translucent, using additive blending for a glow-like effect, and using image textures that already have alpha transparency.


Shadow

Using spotlights to create shadow effects in a scene.
